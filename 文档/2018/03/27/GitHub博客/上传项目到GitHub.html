上传项目到GitHub

1：在终端

如果本地没有.ssh文件，执行下面一行

$ ssh-keygen -t rsa -C "wangtaoios@163.com"

进入.ssh

出现+----[SHA256]-----+即成功

* 此过程要输入一个6位及以上的临时密码，在步骤4的“克隆”之前需要在弹框输入

最后拷贝公钥

$ pbcopy < id_rsa.pub

2：将公钥粘贴添加到GitHub的SSH and GPG keys列表

3：在GitHub点击+号，选择New repository，创建项目名字，拷贝SSH链接

4：在SourceTree选择新建，从URL克隆，粘贴SSH链接，克隆成功后，项目就在本地目录，即可以进行修改，提交，推送
